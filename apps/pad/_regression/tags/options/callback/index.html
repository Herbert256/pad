It's possible to put a PHP file in the directory 'callbacks' that is called from the called back at the processing of a pad tag.

PHP
HTML
tags/options/callback/before.html 		RESULT
<table border=1>

  <tr>
    <th>Name</th>
    <th>Salary</th>
    <th>Bonus</th>
    <th>Total</th>
  </tr>
  
  {staff callback='before.php', before}

    {first}
      <tr>
        <td>Totals</td>
        <td>{$salaryTotal %.2f}</td>
        <td>{$bonusTotal  %.2f}</td>
        <td>{$totalTotal  %.2f}</td>
      </tr>
    {/first}

    <tr align="right" bgcolor="{switch 'yellow', 'pink'}">
      <td align="left">{$name capitalize}</td>
      <td>{$salary %.2f}</td>
      <td>{$bonus  %.2f}</td>
      <td>{$total  %.2f}</td>
    </tr>

  {/staff}
 
</table>
