Include an other page in the current page, 8 ways of doing this.

3 ways that only work in 'include' mode

HTML RESULT
{page 'hello',      type='include', $color='orange'}

Hello World !

{page 'hello/php',  type='php',     $color='red'}

Hello World !

{page 'hello/html', type='html',    $color='blue'}

Hello World !

5 ways that work in both 'include' and 'full' mode

HTML RESULT
{page 'hello', type='build',    include, $color='pink'}

Hello World !

{page 'hello', type='get',      include, $color='red'}

Hello World !

{page 'hello', type='sandbox',  include, $color='purple'}

Hello World !

{page 'hello', type='function', include, $color='orange'}

Hello World !
